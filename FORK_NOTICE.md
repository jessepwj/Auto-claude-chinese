# Fork Notice / 分支说明

## English

### About This Fork

This is a **Simplified Chinese localization** of [Auto Claude](https://github.com/AndyMik90/Auto-Claude), an autonomous multi-agent coding framework.

### Original Project

- **Repository**: [AndyMik90/Auto-Claude](https://github.com/AndyMik90/Auto-Claude)
- **Author**: [@AndyMik90](https://github.com/AndyMik90)
- **License**: AGPL-3.0

### What's Different

This fork maintains the complete functionality of the original Auto Claude while adding:

1. **Complete Chinese Interface** - All UI elements translated to Simplified Chinese
2. **Chinese Documentation** - Full Chinese README and contributing guidelines
3. **Multi-language Support** - Seamless switching between English, French, and Chinese
4. **Continuous Sync** - Regular updates from the upstream repository

### Scope of Localization

**Fully Localized Components:**
- ✅ Welcome Screen & Onboarding
- ✅ Kanban Board & Task Management
- ✅ Task Details & Specifications
- ✅ Agent Terminals
- ✅ Roadmap & Feature Planning
- ✅ Ideation & Discovery
- ✅ Context Management
- ✅ GitHub/GitLab Integration
- ✅ Linear Integration
- ✅ Changelog & Release Notes
- ✅ Settings & Configuration
- ✅ MCP Server Management

**Translation Files:**
- Location: `apps/frontend/src/shared/i18n/locales/zh-CN/`
- Namespaces: `common`, `navigation`, `settings`, `tasks`, `errors`, `onboarding`, `welcome`, `gitlab`, `taskReview`, `terminal`, `changelog`

### Relationship with Upstream

- This is an **independent localization project**, not an official fork maintained by the original author
- We regularly sync with the upstream repository to include the latest features and bug fixes
- All localization work is contributed back to the community
- The original AGPL-3.0 license is preserved

### Update Strategy

1. **Monitor Upstream** - Track new releases from AndyMik90/Auto-Claude
2. **Sync Changes** - Merge upstream updates into this repository
3. **Update Translations** - Translate new features and UI changes
4. **Test Thoroughly** - Ensure Chinese interface works correctly
5. **Release** - Publish updated Chinese version

### Contributing

We welcome contributions to improve the Chinese localization:

- **Translation Improvements** - Better wording, terminology corrections
- **Bug Reports** - Issues specific to the Chinese interface
- **Documentation** - Improve Chinese docs and guides

Please see [CONTRIBUTING_ZH-CN.md](CONTRIBUTING_ZH-CN.md) for contribution guidelines.

### Credits

**Original Project:**
- **Creator**: [@AndyMik90](https://github.com/AndyMik90)
- **Community**: [Auto Claude Community](https://discord.gg/KCXaPBr4Dj)

**Chinese Localization:**
